# Budget App

Monthly budget planner to capture income, expenditures, loans, debt, pocket money, financial planning, and long/short term savings.

## Features

- **Income Tracking** - Gross salary input
- **Expense Categories** - Housing, Transport, Insurance, Communication, Education, Banking, and Occasions
- **Savings** - Emergency, Fixed, Retirement, and Funeral savings
- **Financial Planning** - Education, Electronics, Car, Appliances, Vacation, and more
- **Pocket Money** - Personal spending allowance
- **Debt Management** - Cell phone purchases, Loan settlements, Interest reimbursement
- **Disposable Income** - Auto-calculated (Income minus all outflows)
- **Data Persistence** - Saves per month using localStorage
- **Export** - Download budget as CSV
- **Print** - Print-friendly layout
- **Responsive** - Works on desktop, tablet, and mobile

## Usage

Open `index.html` in a browser. No build tools or server required.

1. Select a month using the date picker
2. Enter amounts for each budget line item
3. Totals and disposable income update automatically
4. Click section headers to collapse/expand categories
5. Use **Export CSV** to download or **Print** to print your budget
